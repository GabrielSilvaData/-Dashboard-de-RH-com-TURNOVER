# -Dashboard-de-RH-com-TURNOVER
Projeto de People Analytics focado na análise de Turnover organizacional. O dashboard apresenta indicadores estratégicos como Headcount, Total de Desligamentos, Turnover médio, análise por tipo de demissão, motivo, departamento e evolução mensal, permitindo identificar padrões e apoiar decisões de RH baseadas em dados.
